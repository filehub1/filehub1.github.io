文件管理，这些困扰你有吗？

🔍 文件找不到？

电脑、手机、平板里的文件散落各处，想用的时候总是找不到。网盘搜索太慢，本地搜索又只能搜一台设备。

📱 多设备访问难？

想用手机查看电脑里的文档，却要先把文件传到网盘。不同设备之间传输文件麻烦又耗时。

👀 预览太麻烦？

想看个文档还要先找对应的软件打开。PDF、图片、代码每种格式都要用不同应用，麻烦！

🐢 搜索太慢？

等搜索结果等到花儿都谢了。文件名稍微不记得，就彻底搜不到。就没有快一点的方法吗？

FileHub 帮你解决

⚡ 即时搜索

输入关键词，结果瞬间出现。支持精确匹配、模糊搜索、正则表达式，满足各种搜索需求。

📄 强大预览

无需下载专用软件，浏览器直接预览。PDF、图片、视频、Office 文档、代码、Markdown，全部内置！

🌐 局域网访问

手机、平板、电脑同一 Wi-Fi 下直接访问。扫二维码就能在手机上浏览电脑文件。

🔒 隐私安全

数据全部本地存储，不上传云端。局域网访问可设置密码保护，安全可控。

🎨 优雅体验

深色/浅色主题适配，暗光环境下也能舒适使用。键盘快捷键熟练后效率翻倍。

🔌 跨平台覆盖

Windows 桌面应用、Android 手机应用、Docker 服务器，总有一款适合你的使用场景。

选择适合你的版本

下载并运行

1. 从 GitHub Releases 下载
2. 解压 zip 文件
3. 运行 filehub.exe

需要 Windows 10/11 及 WebView2 运行时

Releases →源码 →

下载并安装

1. 从 GitHub Releases 下载 APK
2. 在设置中开启"安装未知应用"
3. 打开 APK 文件进行安装

需要 Android 8.0+（API 26+）

Releases →源码 →

Docker 运行

docker pull exiahuang/filehub-server:latest
docker run -d -p 6543:6543 -v /data:/data exiahuang/filehub-server:latest
# 然后在浏览器打开 http://localhost:6543

一条命令即可启动。支持 NAS、树莓派、任意服务器。

Docker Hub →源码 →

安装并运行

npm install -g @exiahuang/filehub
filehub-server

需要 Node.js 18+
